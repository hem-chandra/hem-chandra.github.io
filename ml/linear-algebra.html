From Shallow to Deep

I want to start with the quotes "Mathematics is a Queen Of Science and Linear Algebra is a Queen of Mathematics".. Linear Algebra is widely used in Machine Learning . Linear Algebra is a mathematics of Data, Matrices and Vectors are language of Data (--MachineLearningMastery.com) . In Linear algebra we generally study lines and planes, vector spaces and mappings that are required for linear transforms . The application of linear algebra in computers is often called numerical linear algebra. "numerical" linear algebra is really applied linear algebra. (--Numerical Linear Algebra, 1997. Some Open source and popular numerical linear algebra libraries are:

    Linear Algebra Package, or LAPACK.
    Basic Linear Algebra Subprograms, or BLAS (a standard for linear algebra libraries).
    Automatically Tuned Linear Algebra Software, or ATLAS
    Scipy (Scientific Python)
    Numpy (Numerical Python)

Computers do fast algebric calculations , and today we widely used Graphical Processing Units (GPUs) in ML because of their ability to computer linear algebra operations fast.
